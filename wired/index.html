4 Script

A collection of links is extracted through a Google search. I used the python library degoogle to get the list of search results. The default output of this library has some unnecessary lines, so I removed them using sed. The final output is formatted as links in Org mode's markup language. The stylesheet for this page is taken from orgcss. 

QUERY='"This article appears in the '"$(date +'%B %Y')"' issue." site:wired.com' && \
    degoogle "$QUERY" | \
        sed -e '/^-- [0-9]\+ results --$/ d' -e '/^$/ d' -e 's/^/[/g' -e 's/$/]/g' | \
        tac | paste -d '' - - | tac | \
        sed -e 's/^/[/g' -e 's/$/]/g' -e 's/^/- /g'
